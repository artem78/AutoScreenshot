Notes for Linux users

After unpacking zip you should add execution rights to AutoScreenshot file

chmod +x AutoScreenshot

For save in avif image format you should install libavif-dev package

sudo apt install libavif-dev

If you have "Could not initialize OpenSSL library" error run

sudo apt install libssl-dev
