Title: Features

* Automatic saving screenshots of full screen with specified time interval (from 1 second to 24 hours)
* Saving images in PNG, JPEG, BMP ![PRO version only]({static}/images/pro.png "PRO version only"), TIFF ![PRO version only]({static}/images/pro.png "PRO version only"), <!--or GIF--> WebP ![PRO version only]({static}/images/pro.png "PRO version only") or AVIF ![PRO version only]({static}/images/pro.png "PRO version only") format
* Pause capture when user is inactive (depending on mouse moves and keyboard events) ![PRO version only]({static}/images/pro.png "PRO version only")
* Customizable output filenames with variables (date, time, user, computer name, sequential number) and ability of grouping by folders (for example: by day or month)
* Automatic capture can be started on system startup
* Multiple screens support ![PRO version only]({static}/images/pro.png "PRO version only")
* High DPI support
* Can execute custom command after screenshot was taken ![PRO version only]({static}/images/pro.png "PRO version only")
* Clearing old screenshots ![PRO version only]({static}/images/pro.png "PRO version only")
* Play sound when screenshot taken
* Hotkeys ![PRO version only]({static}/images/pro.png "PRO version only")
* Works on Windows and Linux
